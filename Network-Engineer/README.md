## SHORT-TERM PLAN (April - June 2026, 2 months)

### Month 1 - April: Foundation + CCNA Prep

#### Week 1-2: Strengthen Core Knowledge
- [ ] Review TCP/IP stack, OSI model (leverage current coursework)
- [ ] Master: subnetting (VLSM, CIDR), ARP, DHCP, DNS, NAT/PAT
- [ ] **Practice**: Use **Cisco Packet Tracer** or **GNS3** - lab at least 1-2 hours/day

#### Week 3-4: Routing & Switching Deep Dive
- [ ] Learn: VLAN, STP, EtherChannel, Inter-VLAN routing
- [ ] Routing: Static routes, OSPF (single area + multi-area), BGP basics
- [ ] VPN: Site-to-site VPN, IPSec concepts
- [ ] **Lab**: Build multi-router OSPF + VLAN topology on Packet Tracer/GNS3
- [ ] Start CCNA practice exams (**Boson ExSim** or free question banks)

### Month 2 - May: CCNA + Linux + Start Project

#### Week 1-2: Complete CCNA Knowledge
- [ ] Security fundamentals: ACL, Firewall concepts, AAA
- [ ] Wireless basics, QoS, Network automation concepts
- [ ] **Register for CCNA 200-301 exam** (~$330, schedule for late May or early June)
- [ ] Daily practice exams

#### Week 3-4: Linux + Home Lab Project
- [ ] Install **Ubuntu Server** on VM (VirtualBox/UTM on Mac)
- [ ] Learn: basic commands, file system, user management, networking config, firewall (iptables/ufw)
- [ ] **Start project - Home Lab Network** on GNS3/EVE-NG:
  - Topology: Router (OSPF) -> Firewall -> Switch (VLAN) -> Servers (Linux)
  - Install monitoring: Zabbix or PRTG free
  - Document on GitHub

### Month 3 - June: Pass CCNA + Finalize Project + Apply

#### Week 1-2:
- [ ] **Pass CCNA exam**
- [ ] Finalize home lab project with clean documentation on GitHub

#### Week 3-4: CV + Job Applications
- [ ] Write CV highlighting: CCNA cert, home lab project, coursework knowledge
- [ ] Create GitHub repo with project documentation
- [ ] Start applying to intern/fresher positions (VNG, FPT, Endava)

### Short-term Deliverables Checklist
- [ ] **CCNA 200-301** certification
- [ ] 1 GitHub project: Home Lab Network (GNS3 topology + Zabbix monitoring + Linux servers)
- [ ] Linux basic skills
- [ ] CV ready to apply

## LONG-TERM PLAN (April 2026 - April 2027, 1 year)

### Q1: April - June *(Same as short-term plan above)*
- CCNA + Linux + Home Lab project

### Q2: July - September - Firewall + Cloud + Scripting

#### Firewall & Security (high priority - most JDs require this)
- [ ] Study Fortinet NSE 4 (FortiGate) or Palo Alto PCNSA
- [ ] Lab: Configure firewall rules, VPN, NAT on FortiGate VM (free trial)
- [ ] Integrate firewall into home lab project

#### Cloud Networking (AWS - strongly required by Renesas JD)
- [ ] Pass **AWS Certified Cloud Practitioner** (foundation)
- [ ] Practice: VPC, subnets, security groups, VPN, Transit Gateway
- [ ] Use AWS Free Tier for hands-on labs

#### Python for Network Automation
- [ ] Learn Python basics (2-3 weeks)
- [ ] Libraries: Netmiko, Paramiko, NAPALM
- [ ] Write scripts to auto-backup router/switch configs
- [ ] Add to GitHub portfolio

### Q3: October - December - Advanced Topics + More Projects

#### Advanced Routing (BGP, MPLS)
- [ ] BGP: eBGP, iBGP, route filtering, communities
- [ ] MPLS fundamentals, L3VPN basics
- [ ] Lab on EVE-NG

#### SD-WAN (required by FPT, Vinamilk JDs)
- [ ] Study Cisco SD-WAN (Viptela) or VeloCloud
- [ ] Build SD-WAN lab if possible

#### Project 2: Network Automation Pipeline
- [ ] Python scripts to auto-configure, backup, monitor network devices
- [ ] CI/CD with Git (meets Endava requirement)
- [ ] Document on GitHub

### Q4: January - April 2027 - Advanced Cert + Job-Ready

#### Next Certification (pick 1)
- [ ] **CCNP Enterprise** (ENCOR 350-401) - for Cisco/ISP track
- [ ] **AWS Solutions Architect Associate** - for cloud networking track
- [ ] **Fortinet NSE 4** - for security track

#### Monitoring & Troubleshooting
- [ ] Master: Zabbix, PRTG, or Grafana + Prometheus
- [ ] Wireshark packet analysis

#### Finalize Portfolio
- [ ] 2-3 projects on GitHub
- [ ] Blog/write-ups about labs (LinkedIn articles)
- [ ] Updated CV with certs + projects + skills

### Long-term Deliverables Checklist
- [ ] **CCNA** (cert)
- [ ] **1 advanced cert** (CCNP / AWS SAA / Fortinet NSE4)
- [ ] **Python scripting** for network automation
- [ ] **Linux** intermediate level
- [ ] **Cloud networking** basics (AWS)
- [ ] **2-3 GitHub projects** demonstrating hands-on skills
- [ ] **CV strong enough** to apply for mid-level positions at FPT, Vinamilk, Mobifone, Renesas
